Chatbot catatan dan referensi mengenai Ilmu Hukum Perdata di Indonesia — kumpulan materi, ringkasan, dan sumber yang disusun untuk membantu belajar dan riset.

## Tujuan
Repositori ini dibuat untuk:
- Mengumpulkan ringkasan sumber Ilmu Hukum Perdata dalam format yang mudah diakses.
- Menjadi tempat belajar bersama, berbagi sumber, dan menyusun bahan ajar.

## Batasan
Struktur direktori :
- `materi/` — ringkasan bab, konsep, dan teori.
- `kasus/` — studi kasus dan analisis putusan dan pasal.
- `sumber/` — jdih Mahkamah Agung repo, Peraturango repo dan BpkRi repo

## Cara menggunakan
1. akses langsung di Telegram: buka bot `@pakbos` lalu tanyai tentang hukum perdata
2. Untuk membaca secara lokal:
   - Clone repositori atau download workflow:
     git clone https://github.com/BroBay24/Rag-ilmuHukumPerdata.git
